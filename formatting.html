HTML Text Formatting


HTML contains several elements for defining text with a special meaning.

Example


This text is bold

This text is italic

This is subscript and superscript



HTML Formatting Elements


Formatting elements were designed to display special types of text

  • <b> - Bold text

    • Syntax :- <b> This text is bold </b>

  • <strong> - Important text

    • Syntax :- <strong> This text is important! </strong>

  • <i> - Italic text

    • Syntax :- <i> This text is italic </i>

  • <em> - Emphasized text

    • Syntax :- <em>This text is emphasized </em>

  • <mark> - Marked text

    • Syntax :-Do not forget to buy <mark> milk </mark> today.

  • <small> - Smaller text

    • Syntax :- <small>This is some smaller text.</small>

  • <del> - Deleted text

    • Syntax :- My favorite color is <del> blue </del> red

  • <sub> - Subscript text

    • Syntax:- This is <sub> subscripted </sub> text

  • <sup> - Superscript text

    • Syntax:- This is <sup> superscripted </sup> text.
